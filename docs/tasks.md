# sshell MVP 模块最小可执行任务清单

## 说明
- 目标：按模块拆分为可独立推进、可验收的最小任务。
- 范围：仅覆盖 `docs/proposal.md` 的 MVP（2.1）与 `docs/detailed-design.md` 的模块设计。

## 1. cli 模块
- [ ] 实现命令行入口与参数读取（支持 1..N 个连接串）
- [ ] 实现帮助信息与参数缺失分支
- [ ] 接入参数错误统一返回（退出码 2）
- [ ] 完成参数解析单元测试（合法/缺参/格式错误）

## 2. connection_spec 模块
- [ ] 定义 `ConnectionSpec` 与显示名结构
- [ ] 实现 `用户名:密码@主机[:端口]` 解析
- [ ] 实现字段校验（IPv4/域名、默认端口 22、拒绝 IPv6）
- [ ] 完成解析单元测试（分隔符边界、端口边界、默认值）

## 3. session_manager 模块
- [ ] 定义会话状态与 `SessionId`、会话顺序模型
- [ ] 实现启动批量建会话（由 CLI 初始连接串触发）
- [ ] 实现运行时新建/关闭/前后切换 Tab
- [ ] 实现“关闭当前后自动选择相邻 Tab”规则
- [ ] 实现“最后一个 Tab 关闭触发退出”事件
- [ ] 完成状态机单元测试（连续关闭、首尾切换、空集边界）

## 4. ssh_runtime 模块
- [ ] 定义 `SshClient` 抽象接口（connect/auth/open_shell/read/write/resize/close）
- [ ] 集成 SSH 库完成用户名+密码认证流程
- [ ] 打通输入输出双向转发链路
- [ ] 实现网络/认证/协议错误映射（3/4/5）
- [ ] 基于 mock 完成失败注入测试（超时、认证失败、断线）

## 5. pty_adapter 模块
- [ ] 定义统一 PTY 适配接口（spawn/read/write/resize/close）
- [ ] 实现 Unix 分支（POSIX PTY）
- [ ] 实现 Windows 分支（ConPTY）
- [ ] 增加 Windows Build 17763+ 前置检查
- [ ] 实现 PTY 初始化失败映射（退出码 6）
- [ ] 完成平台适配层接口测试（假实现 + 平台集成验证）

## 6. terminal_engine 模块
- [ ] 定义终端缓冲、光标与渲染接口（feed/render/resize）
- [ ] 实现 ANSI/VT 基础序列处理（颜色、光标、清屏）
- [ ] 实现滚动区与 alternate screen 行为
- [ ] 打通尺寸变化后的渲染一致性
- [ ] 完成转义序列回归测试与全屏程序关键行为测试

## 7. tui_shell 模块
- [ ] 实现 Tab 栏、会话区、状态栏渲染
- [ ] 实现按键输入分流（本地控制命令 vs 透传远端）
- [ ] 实现前缀键状态机（`Ctrl+A` 前缀方案）
- [ ] 实现运行时新建连接串输入框交互
- [ ] 完成输入状态机测试与 UI 快照测试

## 8. error_model 模块
- [ ] 定义统一错误模型与退出码枚举（0~6）
- [ ] 实现 stderr 输出标准格式 `[ERROR] <code>: <message>`
- [ ] 实现多 Tab 首个错误码聚合策略
- [ ] 完成退出码聚合状态机测试（混合成功/失败顺序）

## 9. app core（编排层）
- [ ] 实现启动编排：CLI -> 会话初始化 -> 事件循环
- [ ] 建立 UI 任务、会话任务、协调任务的消息通道
- [ ] 接入 resize 事件广播与会话同步
- [ ] 接入退出决策（用户正常关闭=0，异常聚合=首错码）
- [ ] 完成基础集成测试（单会话、多会话、单 Tab 异常隔离）

## 10. 跨模块收敛任务
- [ ] 建立需求映射检查（每个 MVP 需求至少映射到一个模块任务）
- [ ] 建立最小验收清单（连接、Tab、终端、错误码）
- [ ] 完成三平台冒烟验证（Linux/macOS/Windows 17763+）
